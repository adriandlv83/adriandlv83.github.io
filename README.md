# Proyectos

## Análisis de comportamiento de usuarios Bike-Share (Cyclistic)

Proyecto de análisis de datos enfocado en identificar diferencias de uso entre usuarios ocasionales y miembros anuales de un sistema de bicicletas compartidas.

### Objetivo

El objetivo estratégico es comprender las diferencias entre tipos de usuario casuales y miembros anuales, con el fin de identificar oportunidades que permitan incrementar la conversión a membresias anuales.

### Metodología

El proyecto sigue un flujo completo de análisis de datos:

- Preparación y limpieza de datos
- Transformación y métricas
- Análisis exploratorio
- Identificación de insights
- Recomendaciones estratégicas

##️# Tecnologías utilizadas

- R
- tidyverse
- ggplot2
- Quarto Website
- GitHub Pages

### Datos

Los datos crudos no se incluyen debido a su tamaño y practicas en restricciones de uso.  
Se proporcionan resúmenes procesados necesarios para reproducir los resultados presentados.
